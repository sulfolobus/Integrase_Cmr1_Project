# Integrase Cmr1 Project - PDB Files

## Overview
This repository contains PDB (Protein Data Bank) structure files for the Integrase Cmr1 protein complex studies.

**Note**: Cmr1b represents **Cmr1Î²** (Cmr1 beta subunit)

## PDB Files Directory Structure

### Cmr1b_MJ0770(WT) - Wild Type Variants
- `Cmr1b_MJ0770(WT)_C1_RoseTTAFold2` - RoseTTAFold2 prediction
- `Cmr1b_MJ0770(WT)_AF_Online` - AlphaFold2 online prediction
- `Cmr1b_MJ0770(WT)_AF_Local` - AlphaFold2 local prediction

### Cmr1b_MJ0770_m6 - Mutant m6 Variants
- `Cmr1b_MJ0770_m6_RoseTTAFold2` - RoseTTAFold2 prediction
- `Cmr1b_MJ0770_m6_AF_Online` - AlphaFold2 online prediction
- `Cmr1b_MJ0770_m6_AF_Local` - AlphaFold2 local prediction

### Cmr1b_Q3V4T7 - Query Sequence Variants
- `Cmr1b_Q3V4T7_C1_RoseTTAFold2` - RoseTTAFold2 prediction
- `Cmr1b_Q3V4T7_m5_C1_RoseTTAFold2` - RoseTTAFold2 prediction (m5)
- `Cmr1b_Q3V4T7_m5_AF_Online` - AlphaFold2 online prediction (m5)
- `Cmr1b_Q3V4T7_m5_AF_Local` - AlphaFold2 local prediction (m5)
- `Cmr1b_Q3V4T7_AF_Online` - AlphaFold2 online prediction
- `Cmr1b_Q3V4T7_AF_Local` - AlphaFold2 local prediction

## Prediction Methods

- **RoseTTAFold2**: Advanced structure prediction method
- **AlphaFold2 (AF_Online)**: AlphaFold2 predictions from online servers
- **AlphaFold2 (AF_Local)**: AlphaFold2 predictions run locally

## File Information

All PDB structure files are included in the compressed archive: `Integrase_Cmr1_Project_PDB files.zip`

Extract the archive to access individual PDB files for structural analysis and visualization.
